## Event Budget Tracker Chatbot (EventIQ)

An AI-powered Event Planning & Budget Management Chatbot that helps users plan events, allocate budgets, track expenses, and receive intelligent financial insights — all through natural conversation.

## 🚀 Features

🤖 Conversational AI (ChatGPT-style)

🎯 Event Planning (Wedding, Birthday, Corporate)

💰 Smart Budget Allocation (Auto % Distribution)

📊 Real-Time Expense Tracking

⚠️ Overspending Alerts & Warnings

💡 AI Insights & Saving Tips

📈 Visual Charts (Budget vs Spending)

🧾 Manual Expense Entry Support

🔄 Persistent App State Management

## Example Use Cases
“Plan a wedding for 200 guests with ₹5,00,000”

“Add ₹10,000 catering expense”

“How much budget is left?”

“Suggest ways to reduce costs”
